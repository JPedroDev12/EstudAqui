# 🎓 EstudaAqui

> Uma página simples e direta para conhecer e acessar os nossos melhores cursos online.

---

## 🚀 Sobre o Projeto

O **EstudaAqui** é uma landing page desenvolvida para apresentar nosso catálogo de cursos de forma rápida e intuitiva. O objetivo é permitir que o usuário veja os cursos disponíveis, suas categorias e clique para se inscrever sem complicações.

### ✨ Funcionalidades

* **Vitrine de Cursos:** Exibição dos cursos em destaque com cards organizados.
* **Filtro por Categorias:** Botões rápidos para filtrar os cursos (ex: Tecnologia, Design, Negócios).
* **Formulário de Contato/Inscrição:** Um rodapé ou seção simples para tirar dúvidas.
* **Design Responsivo:** Funciona perfeitamente no celular e no computador.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura da página.
* **CSS3:** Estilização e layout.
* **JavaScript** dinâmica de filtrar os cursos na tela.
